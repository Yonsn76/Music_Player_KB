# Kb Music Player

Kb Music es un reproductor de audio sencillo y ligero diseñado bajo los principios del minimalismo utilitario. Esta aplicacion permite gestionar tu musica mediante la importacion manual de carpetas especificas del dispositivo, garantizando un control total sobre la biblioteca de audio sin procesos de escaneo automaticos innecesarios.

La aplicacion incluye funciones esenciales como busqueda en tiempo real, modo aleatorio, persistencia de datos y controles avanzados en la barra de notificaciones con soporte para navegacion de progreso.

### Tecnologia
Desarrollado de forma nativa para Android utilizando Kotlin.
